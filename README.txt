========================================
🤖 بوت إعداد وإدارة سيرفر الديسكورد
========================================

المميزات:
- أمر !setup لإنشاء جميع الأقسام والقنوات والأدوار تلقائياً
- نظام ترحيب أوتوماتيكي للأعضاء الجدد
- نظام مستويات (XP / Leveling)
- أوامر إشراف (طرد، حظر، تحذير، مسح)
- دور تلقائي للأعضاء الجدد

========================================
📋 خطوات التشغيل (للمبتدئين)
========================================

الخطوة 1: إنشاء بوت في ديسكورد
---------------------------------
1. افتح الرابط: https://discord.com/developers/applications
2. اضغط على "New Application" وسمّه (مثلاً: My Server Bot)
3. اذهب إلى "Bot" من القائمة اليسرى
4. اضغط "Add Bot" ثم "Yes, do it!"
5. تحت "TOKEN" اضغط "Reset Token" ثم "Copy"
6. هذا هو التوكن (يشبه: abc123.xyz.ABC) - لا تشاركه مع أحد!

الخطوة 2: وضع التوكن في البوت
---------------------------------
1. افتح ملف bot.py بالمفكرة (Notepad)
2. ابحث عن هذا السطر:
   BOT_TOKEN = "ضع_التوكن_هنا"
3. استبدل "ضع_التوكن_هنا" بالتوكن الذي نسخته
   مثال: BOT_TOKEN = "abc123.xyz.ABC"

الخطوة 3: دعوة البوت لسيرفرك
---------------------------------
1. في صفحة المطور، اذهب إلى "OAuth2" > "URL Generator"
2. اختر "bot" من SCOPES
3. اختر "Administrator" من BOT PERMISSIONS
4. انسخ الرابط الذي يظهر وافتحه في المتصفح
5. اختر سيرفرك من القائمة واضغط "Authorize"

الخطوة 4: تشغيل البوت
---------------------------------
1. افتح PowerShell (Windows Terminal)
2. اكتب الأوامر التالية:

   cd D:\الديسكورد\discord_bot
   pip install -r requirements.txt
   python bot.py

3. إذا ظهر "✅ البوت متصل" فالبوت شغال!

الخطوة 5: استخدام البوت
---------------------------------
في سيرفر الديسكورد، اكتب الأمر:
   !setup

البوت راح ينشئ لك جميع الأقسام والقنوات والأدوار تلقائياً.

بعدها استخدم:
   !setwelcome #القناة   - لتعيين قناة الترحيب
   !setautorole @الدور   - لتعيين الدور التلقائي
   !setlevel #القناة     - لتعيين قناة المستويات
   !help                 - لعرض جميع الأوامر

========================================
📌 ملاحظات مهمة
========================================
- البوت يشتغل فقط طالما نافذة PowerShell مفتوحة
- عشان يشتغل 24/7، احتاج استضافة (مثل Render.com - مجاني)
- توكن البوت سري جداً - لا ترسله لأحد
- إذا وقف البوت عن العمل، ارجع شغله من PowerShell

========================================
🆓 استضافة مجانية 24/7
========================================
إذا تبي البوت يشتغل 24 ساعة بدون ما تترك جهازك مفتوح:

1. سجل في https://render.com (باستخدام GitHub)
2. ارفع مجلد البوت إلى GitHub
3. في Render، اختر "Web Service"
4. وصله بمستودع GitHub
5. Build Command: pip install -r requirements.txt
6. Start Command: python bot.py
7. أضف متغير بيئة TOKEN بقيمة التوكن الخاص بك
